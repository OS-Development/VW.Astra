Compiling 

If using Visual Studio Command Prompt then: -
cd C:/viewer/AstraViewer/indra
develop.py configure -DLL_FMOD:BOOL=ON -DPACKAGE:BOOL=ON

Use the command prompt if you have more than one Visual Studio on your machine.
==
cd /c/viewer (using Git Bash)
git clone https://github.com/aurora-sim/AstraViewer.git
cd \viewer\AstraViewer\indra
run dev clean
run dev with fmod

Startup VS C++ (Run As Administrator)
Load .sln from file > Project/Solution
make sure C++ is set to ReleaseSSE2 
"prepare" Build Only prepare
THEN
go in astra folder
there is a vc90libs.rar
unzip it and exctract it in "here" it will overwrite the "libraries" folder

SET astrviewer project as startup project should be in bold

right click on astraviewer project
select properties
then build events
clear pre build (remove the info from both lines) <- Express Edition Only

clear post build (remove the info from both lines) <- Express Edition Only
right click on astraviewer project and select properties

Configuration Properties > C++ > Command lines > Additional options

change /Zm1000 /MP /c /Zc:forScope /Oy- 
TO /Zm500 /MP /c /Zc:forScope /Oy-


Apply
F7 (Build Solution)

2nd Compile to build sound/fmod

run dev clean
then run dev with fmod
dev

Just leave VS C++ open or  Run As Administrator.
right click on astraviewer project
select properties
then build events
clear pre build (remove the info from both lines) <- Express Edition Only
clear post build (remove the info from both lines) <- Express Edition Only
right click on astraviewer project and select properties
Configuration Properties > C++ > Command lines > Additional options
change /Zm1000 /MP /c /Zc:forScope /Oy-
TO /Zm500 /MP /c /Zc:forScope /Oy-
Apply
F7 (Build Solution)
